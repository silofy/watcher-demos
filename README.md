# The Watcher · live demos

Two real offensive-security practice runs, replayed in your browser and graded, with zero install. Each one plays the run command by command, then settles into the graded debrief.

- **HTB Abducted** (medium): [▶ Watch it](https://silofy.github.io/watcher-demos/?demo=abducted)
- **THM RootMe** (easy): [▶ Watch it](https://silofy.github.io/watcher-demos/?demo=rootme)

Both are transcribed from public write-ups, with flags and credentials redacted. This repo hosts only the compiled demo site. The Watcher itself is a local-first flight-data-recorder for offensive-security practice.
